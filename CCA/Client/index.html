General

    This application is used to display contacts, and receive alerts from a server component.

    Groups:

    These are the method for managing the contacts in a tidy way. The names for the groups are supplied by the server component.

    You can set an option to have an All group created. This will show all the contacts in one listing, and removing any duplicate contacts based on unique email address.

    Contacts:

    These will be grouped by names as specified by the server component.

    The data for the contacts are not managed by this application. You can add a contact to your personal contacts and then update accordingly if you so desire.

    Depending on the data supplied, you will be able to send an email, call the contact, or use BBM by simply clicking on the representing data element.

    Alerts:

    There are 2 types of alerts that can be sent to you from the server component. Emergency Call and Emergency Notification.

    Emergency Call will give you information regarding the call (date and time, details) and you must Accept or Decline the alert. If you Accept the alert a Calendar entry will be created for reference.

    Emergency Notification will give you information that you will need to Accept to acknowledge you have read it. Nothing more is required for this alert.
